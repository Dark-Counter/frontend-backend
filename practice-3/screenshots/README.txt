Практика 3 — скриншоты для отчёта
====================================

Поместите сюда один файл со скриншотами:

1) Часть 1: не менее 3 скриншотов тестирования API товаров (Практика 2) в Postman.
   Примеры: GET /products, GET /products/1, POST /products (с телом и ответом).

2) Часть 2: не менее 5 скриншотов запросов к выбранному внешнему API в Postman.

Соберите все скриншоты в один документ (PDF или Word), назовите, например:
  Практика-3-скриншоты.pdf
или
  report.docx

После этого добавьте этот файл в репозиторий и сдайте ссылку на репозиторий в СДО.
